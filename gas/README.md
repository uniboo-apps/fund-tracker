# ファンド・ウォッチャー（スマホ版 / Google Apps Script）

PC不要・スマホでURLを開くだけ。Googleのサーバーが毎朝自動でデータ取得＆
下落（押し目以上）になったら登録メールに通知します。

対象：eMAXIS Slim 全世界株式（オルカン）／ eMAXIS Slim 米国株式（S&P500）

---

## セットアップ手順（初回だけ・5分）

### 1. プロジェクトを作る
1. スマホかPCのブラウザで <https://script.google.com> を開く（Googleログイン）
2. 「新しいプロジェクト」
3. 最初からある `function myFunction(){}` を全部消し、
   `Code.gs` の中身を**まるごと貼り付け**て保存（💾）

### 2. 自動チェック＆通知を有効化
1. 上部の関数選択で **`setup`** を選び「実行」▶
2. 初回は権限の確認が出る → 自分のGoogleアカウントを選び「許可」
   - 「このアプリは確認されていません」と出たら
     「詳細」→「（プロジェクト名）に移動」→「許可」
3. これで **毎朝7時に自動チェック** され、押し目以上の下落時にメールが届く

> 通知先は `Code.gs` 冒頭の `ALERT_EMAIL`（既定 e091228@gmail.com）。変えたい場合はここを編集。

### 3. スマホで見れるURLを発行
1. 右上「デプロイ」→「**新しいデプロイ**」
2. 歯車⚙→種類「**ウェブアプリ**」
3. 設定：
   - 説明：任意
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**自分のみ**
4. 「デプロイ」→ 表示された **ウェブアプリURL** をコピー
5. スマホのブラウザでそのURLを開き、**ホーム画面に追加**しておくとアプリ風に使える

## 使い方
- URLを開くだけ。タブでオルカン/S&P500を切替、期間ボタンで拡大
- 「バーゲン度」バッジと「テクニカル指標」で“今の安さ”を確認
- 下落が来ればメールが届くので、それから開けばOK

## バーゲン度の段階（過去最高値からの下落率）
| 下落率 | 表示 | 意味 |
|---|---|---|
| 〜-2% | 高値圏 | 高め。待ち |
| -2〜-4% | 小休止🟢 | 軽い調整 |
| -4〜-7% | 押し目🟡 | 買い検討（★ここからメール通知） |
| -7〜-12% | バーゲン🟠 | 買い増し有力 |
| -12%〜 | 大バーゲン🔥 | 仕込み時 |

## 動作確認・調整メモ
- 今すぐ通知テスト：関数 `testRun` を実行（条件を満たせばメール送信）
- 通知の閾値を変える：`bandOf_` の数値。「`band >= 2`」が通知開始ライン（押し目）
- 通知時刻を変える：`setup` 内の `atHour(7)` を変更して再度 `setup` 実行
- コードを直したら、必ず「デプロイ」→「デプロイを管理」→ 鉛筆✏→「新バージョン」で更新
- Yahoo(指数)がGoogleのIPで弾かれた場合は指数だけ非表示になります（基準価額とバーゲン判定は正常動作）

## データ元
- 基準価額：投信総合検索ライブラリー（Shift-JIS CSV）
- 指数：Yahoo Finance（S&P500=^GSPC、オルカン=ACWI ETF）
